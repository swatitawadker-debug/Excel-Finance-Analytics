# Excel-Finance-Analytics

## Project Overview:
This project simulates a real-world business scenario for AtliQ Hardwares, where Finance data are transformed into actionable insights using Excel, Power Query, Power Pivot, and DAX and analyzes the financial performance using Profit & Loss (P&L) reports across fiscal years, months, and markets to uncover key trends in revenue, cost, profitability insights and market performance.
